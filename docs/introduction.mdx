# What is Clutch?

Clutch is an expressive visual builder powered by modern tech stack that enables web designers to create AI-powered and traditional web experiences that connect to existing backends and services using MCP. Built on **Next.js** and **React**, Clutch provides visual design tools for professional web designers and developers to build websites, agentic experiences, and AI Apps.

## Who Is Clutch For?

### **Agencies**

Agencies use Clutch to build and maintain client front-ends while clients continue using their existing tools:

- **Unified Workflow**: Bring web designers and developers together in one platform with real-time collaboration
- **Client Flexibility**: Connect to clients' existing backends (WordPress, Drupal, Shopify, proprietary systems) via APIs or MCP
- **Maintenance**: Update and maintain sites visually without requiring client teams to learn new tools
- **Multi-Channel Delivery**: Build traditional sites, agentic experiences, and AI Apps from one project using one component library
- **Version Control**: Integrated Git with visual interfaces that designers can understand

### **Web Designers**

Web designers build visually using Clutch's component-based interface:

- **Visual Building**: Drag-and-drop interface with comprehensive design controls (design tokens, classes, selectors, states, CSS panel)
- **Data Binding**: Visually connect data from backends to components without writing code
- **Component Libraries**: Use Clutch UI or custom component libraries created by developers
- **Responsive Design**: Design for desktop, tablet, and mobile with breakpoint-specific controls
- **Real-Time Collaboration**: Work simultaneously with other designers and developers

### **Developers**

Developers extend Clutch by creating components, queries, actions, and helper functions:

- **Code Components**: Build React components using standard patterns that designers can then use visually
- **Queries**: Create data fetching functions that connect to any backend or use MCP for no-code connections
- **Actions**: Build functions for posting data or triggering workflows (also available via MCP)
- **Helper Functions**: Write utilities for data transformation and formatting during binding
- **Standard Tools**: Use TypeScript, npm packages, and normal React/Next.js patterns

### **Client Teams**

End clients' marketing, content, and e-commerce teams continue using their existing tools:

- Clients use their familiar CMSs (WordPress, Drupal), e-commerce platforms (Shopify), or proprietary backends
- Content teams don't need to learn Clutchâ€”they manage content in their existing systems
- Clutch pulls data from these systems to build the front-end experience
- Agencies handle Clutch; clients handle their content and data

## What Can You Build?

### **Agentic Web Experiences**

Build AI-powered experiences within your site:

- **Generative Landing Pages**: Dynamic, personalized landing pages that adapt per visitor to improve SEO/SEM
- **Conversational Commerce**: Chat-based shopping interfaces that increase conversions and basket sizes
- **Conversational Search**: Natural language search that helps users find content
- **Concierge Experiences**: AI assistants that guide users through workflows

### **AI Apps**

Publish projects as MCP servers for embedding in AI agents:

- **ChatGPT Apps**: Embed branded UI and tools directly into ChatGPT
- **Agent Experiences**: Build for any AI agent that supports MCP
- **Unified Components**: Same component library across websites and AI Apps

### **Traditional Sites**

Build high-performance traditional websites:

- **Marketing Sites**: Landing pages, campaign sites, corporate websites
- **E-commerce**: Product catalogs, storefronts, checkout flows
- **Content Platforms**: Blogs, documentation, knowledge bases
- **Business Applications**: Customer portals, dashboards, tools

### **Hybrid Projects**

Combine traditional and agentic experiences in one project:

- Traditional marketing site with generative landing pages on specific routes
- E-commerce site with conversational commerce on product pages
- Corporate site with conversational search
- One codebase, one component library, deployed as both website and MCP server

**Output Options**: Deploy as a website, publish as an MCP server for AI Apps, or export as a versioned component library.

## Comprehensive Features

Clutch provides professional-grade tools for every aspect of web creation:

### **Visual Design Tools**
- **Drag-and-Drop Builder**: Intuitive visual interface for professional designers
- **Component Library**: Access to Clutch UI and custom component libraries
- **Design Tokens**: CSS variables for colors, typography, spacing, borders, shadows
- **Classes & Sub-Classes**: Organize and reuse styles efficiently
- **Selectors & States**: Style hover, focus, active, disabled, and custom data-driven states
- **CSS Panel**: Comprehensive visual CSS controls for precise design

### **Component System**
- **Pre-built Components**: Rich library of layout, typography, media, and interactive components
- **Code Components**: Developers create custom React components for designers to use
- **Composed Components/Variants**: Build component variants and compositions visually
- **Component Libraries**: Install and version control component libraries across projects

### **Data & Integration**
- **MCP Servers**: Connect to any backend, CMS, or service via Model Context Protocol
- **Queries**: Fetch data from APIs, databases, WordPress, Shopify, or any data source
- **Actions**: Post data, trigger workflows, interact with backends
- **Helper Functions**: Transform and format data during binding
- **Visual Binding**: Designers bind data to components visually without code

### **AI & Agents**
- **Built-in Agents**: Generate content, UI, and support conversational interfaces
- **MCP Integration**: Native support for Model Context Protocol
- **Conversational Interfaces**: Build chat and concierge experiences
- **AI App Publishing**: Generate MCP servers for ChatGPT and other AI agents

### **Collaboration & Workflow**
- **Real-Time Collaboration**: Multiple designers and developers working simultaneously
- **Visual Version Control**: Git integration that designers can understand
- **Design-Developer Handoff**: Seamless collaboration between visual designers and developers

### **Output Options**
- **High-Performance Sites**: Deploy to the edge with Next.js optimization
- **MCP Servers**: Publish as servers for AI Apps in ChatGPT
- **Component Libraries**: Export as versioned, installable component libraries
- **Next.js Foundation**: Full access to Next.js App Router and React Server Components
