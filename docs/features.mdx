# Features

Clutch provides professional-grade features for building agentic web experiences, AI Apps, and high-performance traditional sites. Designed for agencies, professional web designers, and developers, Clutch combines powerful visual tools with infinite extensibility.

## AI & Agentic Capabilities

### AI Apps via MCP
Publish your Clutch project as an MCP (Model Context Protocol) server to embed custom UI and tools directly into ChatGPT and other AI agents. Use the same branded components across your traditional site and AI Apps for consistent experiences across all channels.

### Conversational Interfaces
Build sophisticated conversational experiences:
- **Conversational Commerce**: Chat-based shopping that increases conversions and basket sizes
- **Conversational Search**: Natural language search powered by your content
- **Concierge Experiences**: AI-powered assistants that guide users through complex workflows
- **Generative Landing Pages**: Dynamic, personalized landing pages that adapt to each visitor for improved SEO/SEM

### Built-in Agents
Leverage AI agents that understand your components:
- Generate content based on your data and brand voice
- Create UI using your component library
- Support conversational interfaces with context awareness
- Adapt experiences based on user behavior and intent

### MCP Integration
Connect to any backend, CMS, tool, or service using Model Context Protocol:
- No-code connections to third-party services
- Query data from multiple sources simultaneously
- Take actions across different systems
- Build AI Apps that bring all your tools together

## Design System & Styling

### Design Tokens (CSS Variables)
Centralize your design system with CSS variables:
- **Colors**: Primary, secondary, accent colors with full variant control
- **Typography**: Font families, sizes, weights, line heights, letter spacing
- **Spacing**: Consistent margins, padding, and gap scales
- **Borders**: Radius, width, style, and color values
- **Shadows**: Complete elevation system for depth and hierarchy
- **Light/Dark Mode**: Theme-aware tokens that automatically adapt

Update once, propagate everywhere across all channels—traditional sites, agentic experiences, and AI Apps.

### Classes & Sub-Classes
Organize and reuse styles efficiently:
- **Classes**: Create reusable style classes for components
- **Sub-Classes**: Build hierarchies for better organization
- **Global Styles**: Define styles that apply across projects
- **Scope Control**: Component-specific or project-wide styling

### Selectors & States
Comprehensive state styling for professional designs:
- **Hover States**: Visual feedback on mouse interactions
- **Focus States**: Keyboard navigation and accessibility
- **Active States**: Click and press interactions
- **Disabled States**: Inactive element styling
- **Custom Data States**: Style based on component data attributes
- **Pseudo-Selectors**: Full CSS selector support

### CSS Panel
Professional visual CSS controls:
- Comprehensive property panels for all CSS properties
- Visual controls for layout (Flexbox, Grid)
- Typography controls (font, size, weight, spacing)
- Color pickers with opacity and gradients
- Shadow and border builders
- Transform and animation controls
- Direct CSS editing when needed

### Responsive Design
Mobile-first, responsive design tools:
- **Breakpoint System**: Desktop, tablet, mobile, and custom breakpoints
- **Responsive Properties**: Different values per breakpoint for any property
- **Preview Modes**: Test and edit at any screen size
- **Responsive Show/Hide**: Control component visibility per breakpoint

## Data Integration & Binding

### MCP (Model Context Protocol) Support
Native integration with MCP for universal connectivity:
- Connect to any backend, CMS, or service via MCP
- No-code setup for common services
- Query multiple data sources simultaneously
- Take actions across different systems

### Queries
Developers create queries to fetch data from any source:
- **REST APIs**: Connect to any HTTP API
- **GraphQL**: Query GraphQL endpoints
- **WordPress**: Pre-built queries for WordPress REST API
- **Databases**: Direct database access via serverless functions
- **Custom Sources**: Any data source with an API
- Visual data binding for designers

### Actions
Post data and trigger workflows:
- **Form Submissions**: Handle form data and submissions
- **API Calls**: POST, PUT, DELETE to any endpoint
- **Webhooks**: Trigger webhooks on user actions
- **State Updates**: Update application state
- **MCP Actions**: Execute actions via MCP servers

### Helper Functions
Developers create utilities for data transformation:
- Format data for display (dates, prices, numbers)
- Validate and sanitize user input
- Transform API responses to match UI needs
- Calculate derived values
- Reusable logic across queries and actions

### Visual Data Binding
Designers bind data to components without code:
- Drag-and-drop data binding interface
- Filter and transform data visually
- Conditional rendering based on data
- Loop through collections
- Bind to any component property

## Performance & Deployment

### Next.js Foundation
Fully dynamic sites powered by Next.js:
- React Server Components for optimal performance
- Automatic code splitting and optimization
- Streaming SSR for faster time-to-interactive
- Built-in performance optimizations

### Automatic Optimization
Clutch automatically optimizes everything:
- **Images**: Next-gen formats (WebP, AVIF), responsive sizing, lazy loading
- **Fonts**: Subsetting, preloading, variable font support
- **Code**: Tree shaking, minification, dead code elimination
- **Assets**: Compression and optimization
- **HTML**: Streaming and partial hydration

### Edge Deployment
Deploy globally for best performance:
- Global CDN distribution via Vercel or other edge platforms
- Sub-300ms content propagation worldwide
- 99.99% uptime guarantee
- Automatic scaling and failover
- Zero infrastructure management

### Core Web Vitals Excellence
Optimized for Google's metrics out of the box:
- Largest Contentful Paint (LCP) optimization
- First Input Delay (FID) minimization
- Cumulative Layout Shift (CLS) prevention
- Automatic performance monitoring
